# AI-Powered Cloud Cost Anomaly Detector

**Machine Learning (Isolation Forest) for FinOps Optimization**

This repository provides an automated solution for detecting unusual spikes in cloud infrastructure spending. Designed for engineering leaders and FinOps teams, it leverages unsupervised learning to identify cost anomalies that might indicate misconfigurations, DDoS attacks, or runaway processes.

## 🚀 Key Features
- **Unsupervised Learning:** Uses Scikit-learn's Isolation Forest algorithm.
- **FinOps Focused:** Identifies outliers in daily spending patterns.
- **Scalable:** Easily integrates with AWS, GCP, or Azure billing exports.

## 🛠️ Tech Stack
- **Python 3.9+**
- **Scikit-learn** (ML Engine)
- **Pandas/NumPy** (Data Processing)
